        # Prepare the release

        • Fix Codacy (opens new window) & Intellij warnings.
        • Update security.md (opens new window).
        • Write the release notes from issues in milestones (opens new window).
        • Write a substack post (opens new window).
        • Update & merge Gitbook documentation.

        # Create the release with Maven

        You must be using ssh and not https, to switch to ssh, type :

        git remote set-url origin git@github.com:cassandre-tech/cassandre-trading-bot.git

        Start the release with :

        mvn gitflow:release-start

        After choosing the release number, finish the release, push branches and tags, with this command :

        mvn gitflow:release-finish

        TIP

        The following documentation pages should also be updated :

        • why-cassandre/features-and-roadmap
        • learn/database-structure
        • learn/quickstart#create-your-project
        • learn/dry-mode-and-backtesting
        • learn/technical-analysis/create-the-project
        • learn/technical-analysis/backtest-your-trading-strategy

        # Update Github

        Close the corresponding milestone in Github (opens new window).
